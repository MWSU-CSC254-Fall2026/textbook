# CSC 254 – Object-Oriented Programming

*A deeper look into Java, classes, objects, modularity, and real program design.*

Welcome to CSC 254. This course builds directly on the introductory Java knowledge from CSC 187 and develops the skills needed to design real software in a multi-class, object-oriented environment.

The book you are reading is the primary source of written knowledge for the course. Each lesson is intentionally written so that:

- Experienced programmers will find the explanations clear and worthwhile
- Concepts tie together into a unified picture of *what object-oriented programming actually is*
